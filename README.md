# IntelliScan Setup

## Backend Setup

1. Open terminal inside backend folder

2. Create virtual environment

python -m venv venv

3. Activate venv

Windows:
venv\Scripts\activate

4. Install dependencies

pip install -r requirements.txt

5. Run backend

python run_mode.py

> [!NOTE]
> **Model Weights**: The essential model weight (`resnet_model.pth`) is already included in the repository. You do not need to download it separately.

## Frontend Setup

1. Open terminal inside frontend folder

npm install

2. Run frontend

npm run dev
